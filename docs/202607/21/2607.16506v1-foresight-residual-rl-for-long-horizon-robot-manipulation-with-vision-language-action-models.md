## 摘要
视觉-语言-动作（VLA）策略提供了强大的通用操作先验，但在高公差、接触密集的装配任务中经常失败，原因是长时域信用分配和子任务耦合：对于当前技能在几何上成功的状态可能对下游技能来说是脆弱的。我们在冻结的VLA基础策略上展示了这种残差强化学习（RL）的失败模式：恒定的稀疏成功奖励单独改进了每个子任务，但在技能链式使用时几乎没有增益，因为终端状态质量不受控。我们提出了前瞻残差强化学习（Foresight Residual RL），通过使用离线估计的前瞻值——即基于当前子任务终端状态的未来子任务成功概率——来增强每个子任务的稀疏成功奖励，从而优化交接质量。具体而言，我们（i）从基础策略终端状态的图像中训练一个视觉前瞻预测器，使用下游展开统计数据进行标注，以及（ii）通过反向前瞻归纳训练残差策略，将预测器输出作为奖励乘数。在Isaac Gym中的三相扳手式螺母拧紧装配任务（抓取、移动插入、旋转）上，我们的方法实现了85.6%的完整任务成功率，优于标准子任务残差RL（54.5%）和VLA基线，同时保持子任务成功率不变。这些结果强调，提高长时域性能需要塑造每个子任务产生的成功状态，而不仅仅是是否成功。

## Abstract
Vision-Language-Action (VLA) policies offer strong general-purpose manipulation priors, but often fail on tight-tolerance, contact-rich assembly due to long-horizon credit assignment and subtask coupling: a state that is geometrically successful for the current skill can be brittle for downstream skills. We show this failure mode in residual reinforcement learning (RL) over a frozen VLA base policy: constant sparse success rewards improve each subtask in isolation yet yield little or no gain when skills are chained, because terminal state quality is uncontrolled. We propose Foresight Residual RL, which optimizes handoff quality by augmenting each subtask's sparse success reward with an offline-estimated foresight value -- the probability of future subtask success conditioned on the terminal state of the current subtask. Concretely, we (i) train a visual foresight predictor from images of terminal states of the base policy, labeled using downstream rollout statistics, and (ii) train residual policies via backward foresight induction, using the predictor output as a reward multiplier. On a three-phase wrench-based nut-tightening assembly task in Isaac Gym (grasp, move-insert, rotate), our method achieves 85.6% full-task success, outperforming standard subtask residual RL (54.5%) and VLA baselines, while leaving per-subtask success unchanged. These results highlight that improving long-horizon performance requires shaping which successful states are produced at each sub-task, not only whether success occurs.

- **核心问题**：视觉-语言-动作（VLA）策略虽提供通用操作先验，但在高公差、接触密集的长时域装配任务中表现不佳，主要原因在于子任务间的**因果耦合**（causal coupling）：当前子任务在几何上成功的终端状态可能对下游子任务而言是脆弱的，且标准残差 RL 使用恒定稀疏成功奖励仅奖励子任务本身成功，忽略终端状态质量对后续任务的影响，导致链式执行时性能提升甚微。
- **研究动机**：传统做法（独立训练子任务、分布匹配等）只关注子任务成功或分布重叠，未直接优化终端状态对下游成功的预期回报。论文旨在通过**前瞻值（foresight value）**——未来子任务成功概率——来引导子任务残差策略，优化交接状态质量，从而提升完整任务成功率。
- **整体含义**：强调在长时域接触操作中，**“哪些成功状态被产生”比“是否产生成功状态”更重要**，为残差 RL 在 VLA 策略上的应用提供了跨阶段优化方向。

## 2. 论文提出的方法论：核心思想与关键技术

### 核心思想
- 对每个子任务边界，引入 **离线估计的前瞻值**：即给定当前子任务终端状态，下游子任务成功的概率。将前瞻值作为稀疏成功奖励的乘数，训练残差策略，引导其产生更利于下游的终端状态。

### 关键技术细节
1. **残差策略结构**：  
   - 基础策略：微调 π₀ VLA 模型得到每阶段基础策略 π⁰_k。  
   - 残差策略 π^{res}_k：输入仅手腕相机图像（DINOv2+CNN+LSTM+MLP），输出微小修正量，最终动作 a_t = π⁰_k(o_t) + α·π^{res}_k(o_t)，α 很小（0.1），初期零初始化以保持基础行为。

2. **前瞻预测器（Foresight Predictor）p_φ**：  
   - 输入：终端状态图像（DINOv2 CLS 令牌融合+自注意力+线性层+sigmoid）。  
   - 训练数据收集：对每阶段边界（如 Grasp→Insert），用基础策略收集 N 个成功终端状态；对每个状态，用下游策略（已训练的残差策略）进行 K_rep 次 rollout，记录成功次数 κ_i，得到标签 y_i = κ_i/K_rep。  
   - 损失函数：二项负对数似然（Binomial NLL），尊重计数结构。

3. **后向前瞻归纳（Backward Foresight Induction）**：  
   - 从最后阶段 K 开始向前训练：  
     - 阶段 K：用标准任务奖励（β_K）训练残差策略。  
     - 对 k < K：先训练好阶段 k+1 的残差策略（固定），然后利用该阶段策略产生前瞻预测器 p_φ；再训练阶段 k 的残差策略，奖励为 r_t = β_k(s_{t+1}) · p_φ(o_{t+1})。  
   - 优点：下游策略固定，前瞻估计平稳，避免非平稳性问题。

4. **前瞻值近似**：采用一步近似 V_{k+1}(s) ≈ Pr(子任务 k+1 成功 | s, π_{k+1})，当任务链较长时可扩展为多步。

### 算法流程（文字描述）
- 对每个阶段边界：  
  – 收集基础策略的成功终端状态 X。  
  – 对每个终端状态，用下游策略跑 K_rep 次 rollout 得到成功计数。  
  – 训练二项式 NLL 损失的前瞻预测器。  
- 从最后阶段开始，逆序训练每个阶段的残差策略，奖励为（该阶段成功谓词 × 前瞻预测器输出）。  
- 最后阶段无前瞻（或视为 1）。
