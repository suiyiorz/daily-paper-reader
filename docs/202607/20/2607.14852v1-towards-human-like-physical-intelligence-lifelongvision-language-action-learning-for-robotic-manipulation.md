## 摘要
类似于人类顺序学习新任务的自然能力，配备视觉-语言-动作（VLA）模型的机器人在开放世界环境中部署时应具备终身学习能力以学习新任务。然而，大多数最近提出的终身学习模型旨在有效学习当前任务（可塑性）或保持先前任务的高精度（稳定性），而机器人操作模型中可塑性-稳定性的权衡仍未得到充分解决。为应对这一基本挑战，我们提出了一种用于机器人操作的缓存高效终身视觉-语言-动作学习框架（即LifelongVLA），该框架通过双时间尺度适应机制缓解可塑性-稳定性权衡，同时通过缓存高效的重放策略实现低成本的机器人部署。具体而言，我们提出了一种双时间尺度LoRA门控模块，将VLA适应分解为两条轻量级路径：用于可塑性的短期适配器和用于稳定巩固的长期适配器。这些路径通过任务感知门控集成，从而实现对可塑性-稳定性权衡的显式控制。在技能重放阶段，提出了一种缓存高效的随机重放策略，以在不存储完整轨迹的情况下保留更平衡的保留信号。最后，实验表明LifelongVLA优于现有基线，展示了高效的技能扩展、对已学习操作行为的稳健保留，以及在xArm机器人实际部署中减少对重新训练的依赖。

## Abstract
Similar to the natural capabilities of humans to sequentially learn new tasks, robots with Vision-Language-Action (VLA) models should possess lifelong learning ability to learn a new task when deployed in open-world environments. However, most recently proposed lifelong learning models aim to effectively learn the current task (plasticity) or maintain high accuracy on previous tasks (stability), while the plasticity-stability trade-off remains largely unsolved in robotic manipulation models. To address this fundamental challenge, we propose a cache-efficient lifelong Vision-Language-Action learning framework for robotic manipulation (i.e., LifelongVLA), which alleviates the plasticity-stability trade-off with a dual-timescale adaptation mechanism while achieving low-cost robotic deployment with a cache-efficient replay strategy. More concretely, we propose a dual-timescale LoRA gating module to decompose VLA adaptation into two lightweight pathways: a short-term adapter for plasticity and a long-term adapter for stable consolidation. These pathways are integrated via a task-aware gate, enabling explicit control of the plasticity-stability trade-off. In the skill replay phase, a cache-efficient stochastic replay strategy is proposed to preserve more balanced retention signals without full-trajectory storage. Finally, experiments show that LifelongVLA outperforms existing baselines, demonstrating efficient skill expansion, robust retention of learned manipulation behaviors, and reduced reliance on retraining for real-world deployment on an xArm robot.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究动机**：现有的 Vision-Language-Action (VLA) 模型通常采用静态离线训练范式，在固定技能集上训练一次后部署，无法在开放世界环境中持续学习新任务。机器人需要像人类一样具有终身学习能力：在不遗忘已学技能的前提下获取新技能。
- **核心问题**：终身学习面临“塑性-稳定性权衡”（plasticity-stability trade-off）：新任务学习（塑性）可能会覆盖旧任务知识（导致灾难性遗忘），而过度保留旧知识（稳定性）又会限制新技能获取。现有方法（如经验回放、参数隔离等）未能充分解决这一权衡，且存在存储和计算开销大的问题。
- **整体意义**：提出 LifelongVLA 框架，旨在实现 VLA 模型在顺序任务流中的高效终身适应，同时降低重放开销，为机器人实际部署提供可行的终身学习方案。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- 将 VLA 适应分解为两条轻量级 LoRA 路径（短期/长期），并通过任务感知门控显式控制塑性-稳定性权衡。
- 采用缓存高效的随机重放策略，仅存储紧凑的冻结前缀令牌和状态-动作监督，通过重新采样扩散变量生成多样化的重放实例，避免完整轨迹存储和重复编码。

### 关键技术细节
1. **Dual-Timescale LoRA Gating（双时间尺度 LoRA 门控模块）**：
   - 对每个可适应层 ℓ，定义短期 LoRA 更新 `ΔW_sh` 和长期 LoRA 更新 `ΔW_lg`。
   - 短期路径仅由当前任务损失 `L_new` 更新，实现快速塑性；长期路径仅由重放损失 `L_rep` 和蒸馏损失 `L_dist` 更新（学习率更低 `η_lg < η_sh`），实现稳定巩固。
   - 门控上下文 `c` 从冻结的视觉-语言骨干（停止梯度）中计算，避免循环依赖。
   - 门控输出 `α(c) ∈ [0,1]` 控制长期路径权重，最终组合权重为：`ΔcW = (1-α)ΔW_sh + αΔW_lg`，推理只需单次前向。
2. **Cache-Efficient Stochastic Replay（缓存高效随机重放模块）**：
   - **技能级前缀缓存**：对每个新任务，随机选取少量样本（`ct=50`），仅存储停止梯度的前缀令牌 `h̅_pre`、机器人状态 `s` 和动作 `y`，不存储原始图像、语言令牌、后缀令牌等。
   - **随机重放与后缀重算**：重放时重新采样扩散变量 `τ'` 和噪声 `ϵ'`，用当前模型重新计算后缀令牌，使缓存前缀作为记忆锚点，后缀适应模型变化，避免陈旧表示。
   - **掩码新-重放目标**：当前样本和重放样本混合，分别计算归一化损失 `L_new` 和 `L_rep`，并加入蒸馏损失 `L_dist`（用上一任务快照作为教师），总损失 `L_t = L_new + λL_rep + βL_dist`。

### 公式/算法流程（文字说明）
- 初始化：冻结 VLA 骨干 Θ₀，设置短期/长期 LoRA 参数。
- 每来一个新任务 t：
  1. 收集当前任务数据，随机选择 ct 个样本存入缓存（只存前缀+状态-动作）。
  2. 训练时，从当前任务采样 batch，并从缓存中采样重放 batch，混合后前向传播。
  3. 重放样本的扩散变量和噪声重新采样，用当前模型重算后缀，计算重放损失和蒸馏损失（教师为旧模型快照）。
  4. 根据门控上下文计算组合权重，更新短期 LoRA 用当前任务损失，更新长期 LoRA 用重放+蒸馏损失。
  5. 每个任务训练 10,000 步（联合训练 40,000 步）。
