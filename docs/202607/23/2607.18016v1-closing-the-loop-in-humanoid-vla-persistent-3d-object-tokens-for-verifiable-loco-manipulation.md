## 摘要
视觉-语言-动作策略是通用机器人控制的有前景的基础，但长期的人形机器人移动操作要求机器人将任务对象视为跨移动、接触、遮挡和恢复的持久物理实体。我们将这个问题研究为对象状态发散：用于条件化全身动作的对象状态可能与用于判断动作是否达到预期物理关系的状态不同。我们提出了可持久化对象令牌化（POT），它从RGB-D观测中维护角色索引的3D对象记录，并将其转换为用于全身动作专家的对象令牌。实例化为POT-VLA，相同的对象记录条件化动作生成并支持几何谓词检查，产生一个闭环执行系统，其中对象状态既可操作又可验证。在Unitree G1上，POT-VLA在八个真实世界任务族中将匹配的直接GR00T-N1.7基线从39/80成功提升到71/80成功。在一个外部的Being-0对齐参考中，POT-VLA在对齐的服务任务上取得了44/50成功，而Being-0论文报告的成功率为37/50。最大的提升出现在需要维持3D关系的任务上，这表明以对象为中心的持久状态是可验证的人形VLA执行的一个有用的抽象。

## Abstract
Vision-language-action policies are a promising foundation for general robot control, but long-horizon humanoid loco-manipulation requires the robot to treat task objects as persistent physical entities across movement, contact, occlusion, and recovery. We study this problem as object-state divergence: the object state used to condition a whole-body action can differ from the state used to decide whether the action achieved the intended physical relation. We propose \emph{Persistent Object Tokenization} (POT), which maintains role-indexed 3D object records from RGB-D observations and converts them into object tokens for a whole-body action expert. Instantiated as \emph{POT-VLA}, the same object records condition action generation and support geometric predicate checks, yielding a closed-loop execution system in which object state is both actionable and verifiable. On a Unitree G1, POT-VLA improves a matched direct GR00T-N1.7 baseline from 39/80 to 71/80 successes over eight real-world task families. In an external Being-0-aligned reference, POT-VLA achieves 44/50 successes on aligned service tasks, compared with the 37/50 success reported by the Being-0 paper. The largest gains occur on tasks requiring maintained 3D relations, suggesting that persistent object-centered state is a useful abstraction for verifiable humanoid VLA execution.

### 1. 核心问题与整体含义（研究动机和背景）
- **核心问题**：人形机器人在进行长时程移动操作（如整理桌面、递送饮料）时，需要将任务物体视为跨移动、接触、遮挡和恢复的**持久物理实体**。然而现有VLA（视觉-语言-动作）策略通常将物体状态隐式编码在视觉-语言特征中，导致**物体状态发散**：用于生成动作的物体状态与用于验证动作是否成功（如是否抓取成功、是否放置到位）的物体状态不一致。这种不一致会引发抓取错误、放置偏差、虚假完成等失败模式。
- **整体含义**：为了在真实世界实现可验证的长期操作，需要一种**以物体为中心的持久物理记忆**，既能指导动作生成，也能在动作执行后验证物理结果。该论文提出的POT（Persistent Object Tokenization）正是为了解决这一闭环状态表示问题。

### 2. 方法论：核心思想、关键技术细节、算法流程（文字说明）
- **核心思想**：提出**持久物体令牌化（POT）**，从RGB-D观测中提取角色索引的3D物体记录，将其序列化为**持久3D物体令牌**，然后插入到全身动作专家的自注意力序列中。相同的物体记录同时用于动作条件化和几何谓词验证，形成闭环执行系统（maintain–act–observe–verify–recover循环）。
- **关键技术细节**：
  - **物体记录构建**：给定指令和当前观测，通过VLM规划器或人工提供类型化子任务计划，每个子任务定义角色（如目标、目的地、支撑物、交接伙伴）、接地查询、成功谓词等。从RGB-D观测中，利用SAM3分割、深度反投影、坐标变换生成每个物体的记录，包含：语义角色（TARGET/DESTINATION等）、可见性、置信度、2D边界框、3D质心与范围、可选的姿态、以及与机器人末端执行器、其他物体的关系特征（距离、包含、支撑等）。
  - **持久物体令牌化**：物体记录被映射为固定大小（K=8个槽位，每个槽位F=33个特征）的张量。每个有效槽位携带上述字段，未使用槽位填充为PADDING并掩码。所有度量字段归一化到机器人基坐标系。令牌化后的张量经过一个轻量投影网络（LayerNorm → Linear → GELU → Dropout → Linear）嵌入到扩散变换器（DiT）的隐藏维度，并与角色嵌入、上下文嵌入结合。
  - **动作头插入**：POT-VLA将物体令牌插入到DiT动作头的自注意力序列中，布局为 `[状态令牌, 物体令牌, 动作令牌]`，视觉-语言骨干特征用于交叉注意力。动作头预测下一段全身动作块。
  - **闭环执行流程**：
    1. 根据当前子任务和观测，生成物体记录Mτi_t → 序列化为物体令牌。
    2. 动作专家输出候选动作块，经过运行时检查（平滑、限幅、平衡）后执行。
    3. 执行后，RGB-D刷新同一物体记录（Mτi_{t+H}）。
    4. 几何谓词监督器读取刷新后的记录，检查物理关系（如包含、支撑、末端-物体距离、目标-目的地位移等）。若满足完成谓词则子任务完成；否则根据诊断触发重观察、重接地、重试或重新规划。
  - **谓词验证**：每个谓词包含类型、参数物体、比较运算符、阈值、稳定窗口。基于物体记录的几何证据计算是否满足。可见性低或置信度低时标记为uncertain，触发重观察。
- **训练与适配**：使用与基础动作专家相同的动作块预测损失，不添加辅助损失。物体令牌作为侧通道与演示配对进行微调。

### 3. 实验设计：数据集/场景、Benchmark、对比方法
- **平台**：Unitree G1人形机器人，配置Dex3-1灵巧手和头戴RGB-D相机，在杂乱办公室环境执行。
- **8个真实世界任务家族**（每个任务10次试验，共80次）：
  1. 手推车运输与放置
  2. 薯片盒放入灰色篮子
  3. 两个球放入灰色篮子
  4. 堆叠三个杯子
  5. 衣物放入洗衣篮
  6. 抽屉/托盘放置并关闭
  7. 桌面分类（整理）
  8. 近距离饮料递送
- **基准方法**：
  - **直接基线**：使用相同GR00T-N1.7动作专家、相同本体和运行时，但**禁用持久物体记录、物体令牌和几何谓词监督**。
  - **消融变体**：仅验证器（直接动作+谓词检查和恢复）、仅POT令牌（物体令牌条件化+无谓词恢复）、完整POT-VLA。
  - **外部参考**：在5个Being-0对齐的服务任务（取瓶子、递篮子、抓瓶子、放篮子、放咖啡）上，报告Being-0论文中的结果（37/50）与POT-VLA（44/50）对比。
- **泛化测试**（每个设置10次）：新物体实例、布局偏移（20-60cm位移、30-90度旋转）、干扰物、中途扰动。

### 6. 主要结论与发现
- POT-VLA显著提升成功率：从直接基线的39/80（48.75%）提升至71/80（88.75%）。
- 最大增益出现在需要**维持3D关系**的任务（堆叠、两球放置、衣物搬运、抽屉交互），验证了物体状态发散是主要失败原因。
- 消融实验表明：物体令牌条件化贡献最大（从40%提升至77.5%），验证器提供额外增益（77.5%→85%）。两者结合效果最佳。
- 泛化测试中，POT-VLA对新物体实例、布局偏移、中途扰动均表现稳健（成功率80-90%），视觉干扰影响较小。
- 在外部Being-0对齐任务上，POT-VLA达到88%（44/50），超过Being-0报告的74%（37/50）。

### 7. 优点：方法或实验设计的亮点
- **方法优势**：
  - 提出**共享物体记忆**，同时用于动作条件化和几何验证，天然解决状态发散问题。
  - 持久化物体记录支持不确定性表达（可见性、置信度标记），允许系统在低信心时触发重观察，避免错误前进。
  - 令牌化设计兼容现有VLA动作头（仅需在自注意力中插入额外令牌），便于集成。
  - 不依赖仿真器、物理引擎或学习动力学模型，仅基于RGB-D测量。
- **实验优势**：
  - 在真实机器人上评估，任务涵盖运输、放置、堆叠、分类、柔性物体、铰接物体、交接等多样性。
  - 消融设计清晰隔离了“更强感知”和“更强验证”的贡献。
  - 泛化测试覆盖多种干扰，证明鲁棒性。
