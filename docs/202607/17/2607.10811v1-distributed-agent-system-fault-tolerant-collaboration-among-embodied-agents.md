## 摘要
人工智能工程正在从大型语言模型（LLMs）的被动文本生成转向智能体驱动的任务执行，这给资源受限和环境不确定下的长期任务带来了新的可靠性挑战。传统的误差消除优化策略无法解决累积误差传播问题。本文提出分布式智能体系统（DAS），一种用于异构智能体间容错协作的设备-边缘-云框架。我们将智能体可靠性重新定义为系统级容错而非单轮零误差准确率，并提出双层容错架构：通过容错对齐实现单智能体执行可靠性，以及通过半形式化语言协议实现跨智能体通信可靠性。该框架为工业场景中可靠的异构具身智能体协作提供了实用的工程路径。

## Abstract
AI engineering is shifting from passive text generation by large language models (LLMs) to agent-driven task execution, creating new reliability challenges for long-horizon tasks under resource constraints and environmental uncertainty. Conventional error-elimination optimization strategies fail to address cumulative error propagation. This paper proposes Distributed Agent System (DAS), a device-edge-cloud framework for fault-tolerant collaboration among heterogeneous agents. We redefine agent reliability as system-level fault tolerance rather than single-turn zero-error accuracy, and present a two-layer fault-tolerance architecture: single-agent execution reliability via fault-tolerant alignment, and cross-agent communication reliability via semi-formal language protocols. This framework provides a practical engineering pathway for reliable heterogeneous embodied agents collaboration in industrial scenarios.

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
- **不确定性不可消除**：模型和环境引入的不确定性是智能体交互和物理操作的内在属性，不应追求完全消除，而应通过容错管理来抑制误差传播。
- **可靠性再定义**：可靠性是智能体在给定任务和环境约束下，稳定、一致、可控地完成闭环任务过程的能力，本质是任务收益与硬件、时间、人工干预成本之间的效用权衡。
- **双层容错架构**：
  - **第一层：单智能体执行容错**——通过“容错对齐”（Fault-Tolerant Reliability Alignment）让轻量级智能体学会在能力边界内采取容错动作（如主动拒绝、意图澄清、任务路由），而非强行输出错误答案或执行危险操作。
  - **第二层：跨智能体通信容错**——通过半形式化语言协议（Semi-Formal Language Protocols）将自然语言提示中隐含的任务约束、交接规则转化为可检查、可执行、可审计的形式化结构，阻断语义偏差沿协作链级联传播。

### 关键技术细节
- **容错对齐范式**：
  - 扩展智能体的动作空间，在常规任务执行动作之外加入“不确定动作空间”（如拒绝、澄清、引导提问、跨层任务路由）。
  - 采用端到端可训练优化框架（强化学习持续迭代），区分于被动的“置信度检测+固定规则”方法。
  - 实现方式：①显式建模知识边界处的不确定容错动作（通过专门对齐训练）；②DiSRouter（分布式自路由）机制，让智能体自主评估能力并决定本地执行还是向上委托给更高能力智能体。
- **半形式化语言协议**：
  - 基于“表达能力 vs 可验证性”的权衡（参照Chomsky语言层级），纯自然语言灵活但模糊，纯形式语言严格但语义受限。
  - 将隐式规则（任务约束、转换策略、交接职责）转化为标准化的形式协议结构，分离显式约束审计与隐式通用推理，同时保留自然语言的语义推理能力。
  - 双保险机制：编译时进行语法验证、逻辑检查、参数校验；运行时通过守卫机制实时约束智能体交互行为，阻断违规操作。
